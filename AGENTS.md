# General rules
- This is a  research paper for a top-tier journal: all the algorithms and methods should be described in a way that is understandable to the audience. No references to the codebase are allowed.
- Prefer using `\emph{}` over quotation marks ` `` '' `.
- Each paragraph should have a short comment above it describing what the paragraph is about. Example:
    ```LaTeX
    % Define finite element spaces.
    ```
- Avoid tautologies. Examples:
        - *Mathamatically equivalent* -> *equivalent*
        - *rigorously justifies* -> *justifies*
        - *The fundamental objective* -> *The objective*
